Remedy Share

This web app provided a means of sharing your home remedies with people around the globe. Hosted on AWS EC2 instance using MongoDB and Redis as data stores. This is the first project I made for the complete stack - Android, WEB frontend, WEB Backend and DevOps.

Git Repository     Android App
Technologies used: NodeJS MongoDB TinyMCE AngularJS

Why?

This was my first full Android project. It was created mainly for learning purposes. I wanted to created a simple application to start my venture into play store and at the same time my dad used to take medications and he used to tell us to remind him to take the medicines on time. I thought why not create an app for it?

Initially this app was a very simple alarm only app. As the days passed and I got more into android, I started using custom components, changed UI and introduced search APIs. Even a web-app of the same was built which was discontinued due to some logistical issues.

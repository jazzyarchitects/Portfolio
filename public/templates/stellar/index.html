Because practice makes perfect

Haptiq

The prototype built during Hackfest 2017 in 36hrs, Haptiq is a fingerprint based authentication system for social logins. It enables you to login to various social sites using your Android Phone's Biometric authentication system. Your data is kept confidential. The app uses AES 256-bit encryption for communicating with the chrome extension.

Project Repositories     Android App   Chrome Extension   Server
Technologies used: Android NodeJS Socket.IO Chrome Extension

FoodKart

Our Hackfest 2016 project, an android application created with an objective to make ordering food more economical for students. Every restaurant has a minimum order criteria present in order to avail its delivery facilities. We as individuals are forced to satisfy the minimum order criteria even though the order exceeds our appetite. This app pools enables individuals to pool together orders of various individuals together hence benefiting the individuals.

Technologies used: Android Ruby On Rails SQLite MongoDB
